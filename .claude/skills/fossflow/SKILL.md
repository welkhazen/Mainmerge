# FossFLOW Skill

An open-source Progressive Web App (PWA) for creating beautiful isometric diagrams and infrastructure visualizations.

## Overview

FossFLOW is a powerful, web-based diagramming tool for DevOps and infrastructure documentation. Create professional isometric diagrams entirely in your browser with offline support and instant collaboration.

## Features

- **Browser-Based** - No installation required, works in any modern browser
- **Offline Capable** - PWA with full offline functionality
- **Drag & Drop** - Intuitive component-based interface
- **Isometric Visualization** - Professional 3D-style diagrams
- **Export & Share** - Save as JSON or image formats
- **Infrastructure Components** - Pre-built library of DevOps elements:
  - Servers & Databases
  - Cloud Services
  - Networks & Firewalls
  - Applications
  - Storage Systems

## Quick Start

### Create a New Diagram
1. Open FossFLOW in your browser
2. Drag components from the library to the canvas
3. Connect components with relationship lines
4. Save your work locally or export

## Best Practices

1. **Start Simple** - Begin with main components, add details iteratively
2. **Use Consistent Naming** - Clear, standardized component labels
3. **Group Related Systems** - Keep related components together visually
4. **Color Code by Type** - Use colors to distinguish service categories
5. **Document Dependencies** - Show all critical connections

## Keyboard Shortcuts

- `Ctrl/Cmd + S` - Save diagram
- `Ctrl/Cmd + E` - Export
- `Ctrl/Cmd + Z` - Undo
- `Del` - Delete selected component
- `Drag` - Move components
- `Click & Drag` - Create connections
