# UI UX Pro Max Skill

An AI skill that provides design intelligence for building professional UI/UX across multiple platforms and frameworks.

## Features

- **161 Reasoning Rules** for design best practices
- **67 UI Styles** covering modern design patterns
- **Design System Generator** - AI-powered design system generation tailored to your project
- **Pattern Analysis** - Identifies optimal UI patterns for your use case
- **Accessibility Checking** - Validates WCAG compliance
- **Style Recommendations** - Intelligent design suggestions based on project context

## Quick Start

### Generate Design System
Request a design system for your project type and get intelligent recommendations:

```
Generate a design system for: [project type]
Target audience: [description]
Brand guidelines: [guidelines]
```

The skill will provide:
- Recommended design patterns
- Color palettes
- Typography systems
- Component architecture
- Layout guidelines

### Get Design Recommendations

Ask for UI/UX recommendations for specific scenarios:

```
Recommend a UI pattern for: [feature]
Platform: [web/mobile/desktop]
User context: [description]
```

### Check Accessibility

Validate your design against accessibility standards:

```
Check accessibility for: [description]
```

## Best Practices

1. **Always define your target audience** - Design decisions vary by user type
2. **Consider platform constraints** - Web, mobile, desktop have different rules
3. **Plan for accessibility** - WCAG AA compliance should be baseline
4. **Test with users** - Recommendations are starting points, not final designs
5. **Iterate based on feedback** - Design is iterative
